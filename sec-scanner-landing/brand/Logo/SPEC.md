# Logo

## Primary Mark
- **Icon**: Shield (lucide-react: Shield)
- **Color**: #00ff88 (accent green)
- **Minimum size**: 24px height
- **Clear space**: 1x shield height on all sides

## Logotype
- **Font**: Geist Sans, semibold (600)
- **Color split**: "sec" in foreground (#e8e8ed), "‑scanner" in accent (#00ff88)
- **Sizing**: "sec‑scanner" at base size, ".pro" suffix in muted (#6b6b80) at 0.875x

## Variants

| Context | Variant | Size |
|---------|---------|------|
| Sidebar (expanded) | Shield + "sec‑scanner" | 32px shield + text |
| Sidebar (collapsed) | Shield only | 32px |
| Header (landing) | Shield + "sec‑scanner" + ".pro" | 28px shield + text |
| Favicon | Shield only | 32px |
| Mobile header | Shield + "sec‑scanner" | 24px shield + text |

## Rules
- Never rotate or skew the shield
- Never place on light background without inverting
- Never add drop shadows or glow effects to the logo itself
- Always maintain clear space
- The shield glow effect (blur-md) is for interactive hover states only
