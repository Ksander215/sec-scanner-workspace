# END_TO_END_CHECKLIST.md — Сценарии проверки

> Каждый сценарий проверяет конкретную страницу от открытия до завершения действия.

---

## Dashboard (/app/dashboard)

### Что открыть
1. Перейти на https://sec-scanner.pro/app/dashboard

### Что нажать
1. Кликнуть на вкладку «Overview»
2. Кликнуть на вкладку «Findings»
3. Кликнуть на вкладку «Compliance»
4. Кликнуть на вкладку «AI Copilot»
5. Кликнуть на кнопку «Что это?» рядом с Risk Score
6. Кликнуть на ссылку «Запустить первую проверку»
7. Кликнуть на AIS плавающую кнопку (правый нижний угол)

### Что должно произойти
- Заголовок «Executive Dashboard» виден
- Бейдж профиля AIS отображается («Определяем ваш профиль...»)
- Вкладки переключаются без перезагрузки
- «Что это?» показывает объяснение Risk Score
- «Запустить первую проверку» ведёт на Scanner
- AIS панель открывается с 3 вкладками
- Метрики (Security Score, Critical, Attack Paths) отображаются

### Что обязательно проверить
- [ ] Security Score показывает число (0-100)
- [ ] Critical count > 0 (demo данные)
- [ ] Compliance Status показывает прогресс
- [ ] Top Recommendations сортированы по impact
- [ ] Personal Goal Card виден с прогрессом
- [ ] AIS контекст соответствует странице Dashboard

## AIS (на всех страниц)

### Что проверить
1. Плавающая кнопка видна в правом нижнем углу
2. Клик открывает панель AIS
3. Вкладка «Навигация» показывает контекстную подсказку
4. Вкладка «Цель» показывает персональную цель
5. Вкладка «Уверенность» показывает оценку
6. Закрытие панели работает
7. Панель не блокирует основной контент

### На разных страницах
- [ ] Dashboard: AIS подсказывает о первой проверке
- [ ] Scanner: AIS подсказывает о сканировании
- [ ] Marketplace: AIS помогает выбрать инструменты
- [ ] Reports: AIS предлагает отправить отчёт
- [ ] Settings: AIS не навязчив
