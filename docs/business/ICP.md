## ICP-1: Startup Founder (CTO/CEO tech startup, 10-50 сотрудников)

### Профиль
- **Кто**: Основатель технологического стартапа, часто технический бэкграунд
- **Компания**: SaaS, fintech, healthtech, devtools — 10-50 сотрудников
- **Контекст**: Нет выделенной security команды, но compliance требуется (SOC2 для enterprise клиентов)
- **Бюджет**: $99-$499/мес из personal budget или seed funding

### Боли
1. "Enterprise клиенты требуют SOC2 — у нас нет security команды"
2. "Я не понимаю, насколько мы безопасны"
3. "Нанимать security engineer — $150k/год, не могу позволить"
4. "У нас 5 разных инструментов (Snyk, Dependabot, Sentry) — хаос"
5. "Инвесторы спрашивают про security — мне нечего ответить"

### Что покупает
- **Professional план** ($499/мес): AI Assistant, регулярные сканы, отчёты для compliance
- **Add-on**: Custom integrations с их stack

### Почему покупает
- AI Assistant говорит на языке бизнеса, не техники
- Один инструмент вместо пяти
- Отчёты готовые для SOC2 аудиторов
- Цена в 30× дешевле security engineer

### Цена
- $499/мес = $6k/год
- LTV: $18k (3 года retention)
- CAC: $300 (LinkedIn outreach + community)

### Путь пользователя
1. LinkedIn пост о security для стартапов → клик
2. Landing → "Try free for 14 days"
3. Регистрация (email) → onboarding wizard
4. First scan (guided by AI Assistant)
5. Executive Summary → показывает CEO "вот你的 security status"
6. Передача отчёта инвесторам → "у нас есть security platform"
7. Conversion в paid после trial

## ICP-2: SMB CTO (50-200 сотрудников)

### Профиль
- **Кто**: CTO средней компании, часто с командой 2-5 разработчиков
- **Компания**: E-commerce, SaaS B2B, маркетплейсы
- **Контекст**: 1 security engineer (perfiremant), или никто
- **Бюджет**: $499-$2499/мес из IT budget

### Боли
1. "У нас 200 серверов — не успеваем проверять вручную"
2. "Security engineer тратит 60% времени на рутинные сканы"
3. "После инцидента 2023 года — руководство требует регулярный audit"
4. "Compliance: GDPR, PCI-DSS — нужны доказательства регулярных проверок"

### Что покупает
- **Business план** ($1499/мес): безлимит сканов, team features, API access
- **Add-on**: Onboarding для всей команды

### Почему покупает
- Автоматизация рутины security engineer'а
- Team features: shared dashboards, RBAC
- API для интеграции с их CI/CD
- Compliance-ready отчёты

### Цена
- $1499/мес = $18k/год
- LTV: $54k (3 года)
- CAC: $800 (content marketing + webinars)

### Путь пользователя
1. Google search "vulnerability scanner for SMB" → SEO статья
2. Landing → "Book a demo"
3. Demo с sales (30 мин)
4. Trial с full features
5. Onboarding команды (5 пользователей)
6. Integration с их stack (GitHub, Slack, Jira)
7. Conversion после успешного trial

## ICP-3: Enterprise CISO (500+ сотрудников)

### Профиль
- **Кто**: Chief Information Security Officer крупной компании
- **Компания**: Bank, insurance, healthcare, telecom
- **Контекст: Есть security команда 5-20 человек, но нуждаются в автоматизации
- **Бюджет**: $2499-$9999/мес из security budget

### Боли
1. "У нас 10 разных security инструментов — каждый стоит $50k+/год"
2. "Consolidate security stack — стратегическая цель"
3. "SSO, RBAC, audit trail — mandatory для enterprise"
4. "On-premise опция — для regulated industries"

### Что покупает
- **Enterprise план** ($4999/мес): SSO, RBAC, audit trail, on-premise, dedicated support
- **Custom**: Professional services для интеграции

### Почему покупает
- AI Assistant экономит время security команды на analysis
- Consolidation: один инструмент вместо пяти
- Enterprise features: SSO, RBAC, audit trail
- On-premise для regulated industries

### Цена
- $4999/мес = $60k/год
- LTV: $180k+ (3 года, expansion revenue)
- CAC: $5000 (enterprise sales, длительный цикл 3-6 мес)

### Путь пользователя
1. Industry conference / webinar → знакомство
2. White paper "AI for Enterprise Security" → download
3. Sales outreach (LinkedIn, email)
4. Discovery call (60 мин)
5. Technical evaluation (POC, 30 дней)
6. Security review (их security команда проверяет нас)
7. Legal review (MSA, DPA, SLA)
8. Contract signing
9. Onboarding + training

## ICP-4: Security Agency / Consulting Firm

### Профиль
- **Кто**: Владелец security консалтинговой компании
- **Компания**: 5-30 сотрудников, оказывает security audit услуги
- **Контекст**: Проводит аудиты для 10-50 клиентов в год
- **Бюджет**: $999-$4999/мес из operating budget

### Боли
1. "Каждый audit — manual работа, 40+ часов"
2. "Клиенты хотят красивые отчёты, а не Excel таблицы"
3. "5 консультантов — каждый использует свои инструменты"
4. "Масштабирование: не могу взять больше клиентов без найма"

### Что покупает
- **Business план** ($1499/мес): multi-client management, white-label отчёты
- **Add-on**: API для автоматизации audit workflow

### Почему покупает
- Автоматизация audit — 40 часов → 8 часов
- White-label отчёты с branding клиента
- Multi-client management в одном dashboard
- Масштабирование без найма

### Цена
- $1499/мес = $18k/год
- LTV: $36k (2 года, churn выше из-за малого бизнеса)
- CAC: $400 (referral от existing клиентов + community)

### Путь пользователя
1. Referral от другого консультанта
2. Landing → "For Security Agencies" → case study
3. Demo с focus на multi-client features
4. Trial с 3 pilot клиентами
5. ROI calculation: "экономия 32 часов/мес × $100/час = $3200/мес"
6. Conversion

## ICP-5: MSP (Managed Service Provider)

### Профиль
- **Кто**: Технический директор MSP (IT outsourcing)
- **Компания**: 20-100 сотрудников, обслуживает 50-200 SMB клиентов
- **Контекст**: Добавляет security services к своему offering
- **Бюджет**: $2499-$9999/мес (reseller model)

### Боли
1. "Клиенты спрашивают про security — нечего предложить"
2. "5 разных vendors для security — сложно управлять"
3. "Маржа на security services — нужна white-label platform"
4. "Multi-tenant: каждый клиент изолирован"

### Что покупает
- **Enterprise план** ($4999/мес): multi-tenant, white-label, reseller pricing
- **Revenue share**: 30% margin на resell

### Почему покупает
- New revenue stream: security services для existing клиентов
- White-label: brand как свой сервис
- Multi-tenant: изоляция клиентов
- Reseller pricing: 30% margin

### Цена
- $4999/мес = $60k/год
- LTV: $180k (3 года + resell revenue)
- CAC: $2000 (channel sales, partnerships)

### Путь пользователя
1. Channel partner outreach
2. Landing → "For MSPs" → reseller program
3. Partner agreement + commercial terms
4. Technical onboarding (multi-tenant setup)
5. Sales enablement (collateral, training)
6. First 5 клиентов onboarded
7. Monthly resell revenue

## North Star Metric

**Weekly Active Scans (WAS)** — количество сканирований в неделю.

Почему: WAS коррелирует с value delivery (пользователь получает value когда сканирует), и с retention (регулярные сканы = регулярная ценность).

Цель 2026 Q4: 100 WAS
Цель 2027 Q2: 1000 WAS
Цель 2027 Q4: 5000 WAS
