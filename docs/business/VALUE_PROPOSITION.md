## Core Value Proposition

**SIP — Security Intelligence Platform для бизнеса без security команды.**

Понимайте, контролируйте и улучшайте безопасность без найма $150k/год engineer.

---

## Почему клиент покупает именно нас?

### 1. AI Assistant, который говорит на языке бизнеса
- **Конкуренты**: Tenable, Qualys, Rapid7 — технический язык (CVE, CVSS, NIST)
- **Мы**: "Найдена критическая уязвимость. Риск: утечка данных. Время на исправление: 2 часа."
- **Value**: CEO понимает за 30 секунд, без переводчика

### 2. Один инструмент вместо пяти
- **Конкуренты**: Snyk (code) + Tenable (infra) + Dependabot (deps) + Sentry (errors) + manual reports
- **Мы**: Одна платформа — code + infra + deps + reports + AI
- **Value**: $50k+/год экономии на tool consolidation

### 3. Compliance-ready отчёты из коробки
- **Конкуренты**: Custom reports требуют 40+ часов consultant time
- **Мы**: Executive Summary на 1 странице, готовая для SOC2 аудиторов
- **Value**: 40 часов × $100/час = $4000 экономии на каждом audit

### 4. Цена в 30× дешевле security engineer
- **Конкуренты**: $150k/год engineer + $50k tools = $200k/год
- **Мы**: $499-$1499/мес = $6k-$18k/год
- **Value**: $180k+/год экономии

### 5. First Value за 5 минут
- **Конкуренты**: Setup 1-2 недели, первый scan через месяц
- **Мы**: Регистрация → first scan → first report за 5 минут
- **Value**: Time-to-value в 1000× быстрее

## Почему не конкурента?

### vs Tenable ($50k+/год)
- **Их преимущество**: Глубина vulnerability database, enterprise features
- **Наше преимущество**: 10× дешевле, AI Assistant, setup за минуты
- **Кто выберет нас**: SMB 50-500 сотрудников без security команды

### vs Snyk ($7k-$50k/год)
- **Их преимущество**: Deep code analysis, developer-focused
- **Наше преимущество**: Не только code, но и infrastructure + reports
- **Кто выберет нас**: Компании, которым нужна полная картина, не только code

### vs Rapid7 ($100k+/год)
- **Их преимущество**: Full security platform, MDR services
- **Наше преимущество**: AI Assistant, цена, simple UX
- **Кто выберет нас**: Компании, которые не могут позволить Rapid7

### vs Manual security audits ($10k-$50k per audit)
- **Их преимущество**: Глубокий human analysis
- **Наше преимущество**: Регулярность (не раз в год), цена, AI
- **Кто выберет нас**: Компании, которым нужны регулярные проверки, не разовые

## Что невозможно получить нигде ещё?

### 1. AI Assistant, который знает ваш контекст
- Запоминает вашу инфраструктуру, роль, историю
- Предлагает следующие шаги, не только информацию
- Адаптируется к скорости чтения, предпочтениям

### 2. Business-язык в security
- Каждый finding переведён на язык бизнеса (стоимость, риск, время)
- Executive Summary для CEO без технических терминов
- AI объясняет "почему это важно" и "что делать"

### 3. First Value за 5 минут
- От регистрации до первого отчёта — 5 минут
- Никакой другой security tool не даёт такой скорости

### 4. Consolidation: 1 вместо 5
- Code + Infrastructure + Dependencies + Reports + AI в одной платформе
- Ни один конкурент не покрывает все 5 областей

## Конкурентные преимущества

### Defensible Moats

1. **AI Training Data**: Чем больше пользователей, тем лучше AI Assistant. Сетевой эффект.
2. **Context Memory**: AI помнит контекст каждого пользователя. Перейти к конкуренту = потерять контекст.
3. **Compliance Reports Library**: Накопленная библиотека report templates для разных стандартов.
4. **Integrations Ecosystem**: 50+ integrations = высокий switching cost.
5. **Brand Trust**: "Security platform, которой доверяют" — сложно построить, легко потерять.

### Temporary Advantages (нужно укреплять)

1. **AI Assistant UX**: Сейчас лучший в классе, но конкуренты догонят
2. **Цена**: Сейчас ниже рынка, но нужно поддерживать
3. **Speed**: First Value за 5 минут — нужно сохранять

## North Star Metric

**Weekly Active Scans (WAS)** — еженедельные активные сканирования.

### Почему WAS?
- Коррелирует с value delivery (пользователь получает value когда сканирует)
- Коррелирует с retention (регулярные сканы = регулярная ценность)
- Коррелирует с revenue (больше сканов = выше план)
- Легко измерять

### Цели
- Q4 2026: 100 WAS
- Q1 2027: 300 WAS
- Q2 2027: 1000 WAS
- Q3 2027: 2500 WAS
- Q4 2027: 5000 WAS

### Расчёт revenue
При 1000 WAS × $499/мес (avg) × 80% paid = $400k MRR = $4.8M ARR (цель Q2 2027)
