# Nuclei — Плагин поиска уязвимостей по шаблонам

Плагин **Nuclei** выполняет автоматизированное сканирование целевого ресурса с использованием обширной библиотеки шаблонов ProjectDiscovery. Шаблоны покрывают тысячи CVE, ошибки конфигурации, раскрытие чувствительных данных и другие классы уязвимостей. Сканер поддерживает работу с HTTP, DNS, TCP и другими протоколами, что делает его универсальным инструментом для пентеста. Результаты выводятся в формате JSON и классифицируются по уровню критичности — от critical до info. Плагин идеально подходит для быстрой проверки известных уязвимостей и регулярного мониторинга безопасности веб-приложений и сервисов.
