# Nikto — Плагин сканирования веб-серверов

Плагин **Nikto** выполняет комплексное сканирование целевого веб-сервера на наличие известных уязвимостей, опасных файлов, устаревших компонентов и ошибок конфигурации. Nikto проверяет сервер на наличие более 6700 потенциально опасных файлов и программ, устаревших версий серверного ПО, проблем с заголовками безопасности и конфигурационных ошибок. Инструмент поддерживает проверку SSL-сертификатов, обнаружение CGI-скриптов и тестирование методов HTTP. Результаты сканирования предоставляются в формате JSON и интегрируются в единый отчёт SIP. Плагин рекомендуется для первичного аудита веб-серверов и проверки соответствия базовым стандартам безопасности.
