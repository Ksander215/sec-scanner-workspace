# Trivy — Плагин сканирования контейнеров и файловых систем

Плагин **Trivy** предназначен для выявления уязвимостей в Docker-образах, контейнерах, файловых системах, репозиториях Git и Kubernetes-кластерах. Trivy использует актуальные базы данных CVE и поддерживает сканирование пакетов ОС (Alpine, Debian, Ubuntu и др.), а также зависимостей языков программирования (npm, pip, Maven и др.). Помимо уязвимостей, Trivy обнаруживает ошибки конфигурации в Dockerfile, Kubernetes-манифестах и Terraform-файлах. Результаты выводятся в формате JSON с детальной информацией о каждом обнаружении, включая вектор CVSS и версию исправления. Плагин является ключевым инструментом для обеспечения безопасности контейнерной инфраструктуры.
