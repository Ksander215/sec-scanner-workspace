# Semgrep — Плагин статического анализа кода (SAST)

Плагин **Semgrep** выполняет статический анализ исходного кода целевого проекта с использованием правил Semgrep. В отличие от традиционных SAST-инструментов, Semgrep сочетает скорость простых grep-подобных проверок с точностью семантического анализа, что позволяет выявлять уязвимости без ложных срабатываний. Поддерживаются более 30 языков программирования, включая JavaScript, TypeScript, Python, Go, Java и другие. Результаты анализа возвращаются в формате JSON и включают точное расположение проблемы в коде, описание уязвимости и ссылку на документацию. Плагин идеально подходит для внедрения проверки безопасности в процесс CI/CD.
