# Nmap — Плагин сетевого сканирования

Плагин **Nmap** предназначен для обнаружения хостов, открытых портов и служб в целевой сети. Он использует возможности сетевого сканера Nmap с включёнными скриптами обнаружения уязвимостей (`vuln` NSE-скрипты), что позволяет не только перечислить доступные сервисы, но и выявить известные слабые места на найденных узлах. Результаты сканирования возвращаются в формате XML и автоматически разбираются платформой SIP для построения отчёта с классификацией по уровням критичности: critical, high, medium, low и info. Плагин требует указания целевого хоста или сети и рекомендуется в качестве первого шага при проведении аудита безопасности инфраструктуры.
