# OWASP ZAP — Плагин динамического анализа веб-приложений

Плагин **OWASP ZAP** интегрирует один из самых популярных инструментов динамического тестирования безопасности (DAST) в платформу SIP. ZAP выполняет сканирование целевого веб-приложения методом «чёрного ящика»: паук-краулер обходит доступные страницы, а активный сканер проверяет обнаруженные точки входа на наличие уязвимостей из списка OWASP Top 10 — XSS, SQL-инъекции, CSRF, небезопасные заголовки и многие другие. Результаты сканирования предоставляются в формате JSON с детальной информацией о каждом обнаружении, включая запрос, ответ и рекомендации по устранению. Плагин рекомендуется для комплексного аудита веб-приложений.
