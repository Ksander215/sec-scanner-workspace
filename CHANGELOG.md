# Журнал изменений SIP

Все значимые изменения проекта **SIP — Security Intelligence Platform** документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
а версионирование следует [Semantic Versioning](https://semver.org/lang/ru/).

---

## [v0.4.0] — 2025-04-01

### Добавлено

- **Бизнес-отчёты**: генерация профессиональных отчётов для руководства и совета директоров. Поддержка шести форматов экспорта — PDF, DOCX, XLSX, JSON, CSV, HTML. Каждый отчёт включает резюме рисков, графики трендов и рекомендации по приоритизации.
- **Executive Dashboard**: специализированная панель для C-level руководителей с агрегированными метриками безопасности, индексом цифровой устойчивости и сравнением с отраслевыми бенчмарками.
- **Guided Experience**: интерактивный онбординг для новых пользователей с пошаговыми подсказками, обучающими турами и контекстной справкой. Система адаптируется под роль пользователя — аналитик, руководитель или DevOps-инженер.
- Шаблоны отчётов для соответствия стандартам ISO 27001, SOC 2, PCI DSS.
- Модуль расчёта ROI инвестиций в кибербезопасность с визуализацией стоимости инцидентов.

### Изменено

- Улучшена производительность Knowledge Graph при работе с графами более 10 000 узлов.
- Обновлены зависимости фронтенда до Next.js 15 и React 19.
- Оптимизирована загрузка дашборда — время первого рендеринга сокращено на 40%.

### Исправлено

- Устранена утечка памяти при длительных сессиях сканирования.
- Исправлена некорректная сортировка уязвимостей по CVSS-оценке.
- Решена проблема с отображением кириллических символов в PDF-отчётах.

## [v0.3.0] — 2025-03-01

### Добавлено

- **Knowledge Graph (Граф знаний)**: интерактивная визуализация связей между активами, уязвимостями и угрозами. Поддержка фильтрации, масштабирования и трассировки путей атак. Граф строится автоматически на основе результатов сканирования и данных из внешних источников угроз.
- **Attack Paths (Пути атак)**: моделирование цепочек атак на основе фреймворка MITRE ATT&CK. Автоматическое построение графов атак с оценкой критичности каждого шага. Визуализация от точки входа до целевого актива с расчётом совокупного риска.
- **AI Recommendations**: интеллектуальные рекомендации по устранению уязвимостей на основе машинного обучения. Система анализирует контекст инфраструктуры, историю инцидентов и лучшие практики для формирования персонализированных советов с приоритизацией.
- Интеграция с MITRE ATT&CK — маппинг обнаруженных уязвимостей на тактики и техники фреймворка.
- Экспорт графов атак в формате SVG и PNG для включения в отчёты.

### Изменено

- Переработан UI дашборда — добавлены виджеты Knowledge Graph и Attack Paths.
- Улучшен алгоритм скоринга уязвимостей с учётом контекста окружения.
- Расширена локализация — добавлена поддержка китайского языка (zh-CN).

### Исправлено

- Исправлена ошибка дублирования узлов в Knowledge Graph при повторном сканировании.
- Устранена проблема с таймаутом при сканировании больших сетей (более 1000 хостов).

## [v0.2.0] — 2025-02-01

### Добавлено

- **Plugin System (Система плагинов)**: полноценная архитектура расширений с манифестом, API-хуками и изолированной средой выполнения. Плагины могут добавлять новые типы сканеров, парсеры отчётов, визуализации и интеграции с внешними сервисами. Поддержка горячей загрузки без перезапуска сервера.
- **Расширение Marketplace**: увеличен каталог доступных плагинов и коннекторов. Добавлены категории: интеграции (Jira, Slack, Telegram), дашборды (Security Overview, Compliance Tracker), шаблоны отчётов (SOC 2, ISO 27001), AI-промпты и темы оформления.
- **i18n (Интернационализация)**: полная поддержка русской и английской локализации. Переключение языка в реальном времени без перезагрузки страницы. Система извлечения строк для перевода и управления словарями.
- Менеджер плагинов с интерфейсом установки, обновления и удаления.
- API для разработки сторонних плагинов с документацией и примерами.

### Изменено

- Рефакторинг системы маршрутизации бэкенда для поддержки плагинов.
- Улучшена обработка ошибок при установке плагинов из Marketplace.
- Оптимизирован бандл фронтенда — уменьшен размер на 25%.

### Исправлено

- Исправлена проблема с кодировкой UTF-8 в ответах API.
- Устранён конфликт версий зависимостей при установке плагинов.

## [v0.1.0] — 2025-01-15

### Добавлено

- **Landing Page**: основная посадочная страница проекта с описанием возможностей, секцией преимуществ, демонстрацией платформы, тарифами и формой обратной связи. Адаптивный дизайн для мобильных устройств и планшетов.
- **Dashboard**: главный дашборд безопасности с виджетами статистики уязвимостей, графиками трендов, списком критических находок и индикаторами состояния инфраструктуры. Настраиваемая компоновка виджетов с сохранением пользовательских предпочтений.
- **Scanner**: модуль сканирования безопасности с поддержкой нескольких типов сканеров — сетевое сканирование (Nmap), анализ веб-приложений (OWASP ZAP), проверка зависимостей (npm audit, Trivy). Управление очередью задач, мониторинг прогресса в реальном времени, автоматический парсинг и нормализация результатов.
- **Marketplace (базовая версия)**: каталог расширений и интеграций с категоризацией, поиском и предпросмотром. Начальный набор из 12 плагинов и коннекторов.
- Система аутентификации и управления проектами.
- Базовая документация API и руководство по началу работы.

### Известные проблемы

- Сканирование более 500 хостов может приводить к таймауту.
- PDF-экспорт отчётов не поддерживает кириллицу в заголовках.
